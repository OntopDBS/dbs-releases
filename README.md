# DBS Releases

This repository hosts the auto-update channel for **DBS** (Dominos Business Solutions).

It contains **no source code** — only signed installer binaries and
the `latest.yml` manifest consumed by `electron-updater` running inside
the DBS desktop application.

- Source code: https://github.com/OntopDBS/dbs-v2
- Releases:    https://github.com/OntopDBS/dbs-releases/releases
- Vendor:      Dominos Business Solutions (Tunisia)

Releases are published automatically by `scripts/dbs-publish-release.mjs`
in the source repo. Do not push code here.
